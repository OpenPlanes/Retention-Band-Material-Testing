# Retention-Band-Material-Testing
Material durability study of elastic retention bands used in robotic storage systems. Developed a test plan comparing TPU, EPDM, and spandex-based bands using cyclic fatigue, creep, tensile, and environmental testing to evaluate long-term stretch, failure modes, and material performance.
